what is the internet

Ip addresses and identifying people
- can identify people by their ip address, and mac address (media access control)
- Ip - internet protocol
- identifying a host on a network, split in 4 octets. 
ip addresses change from device to device, but cant be active at the same time more than once in the same network. 
- each device has a private and public ip. public is from isp. pricate can be used to communicate with other devices. 

IPv4 - numbering stystem of 2^32 IP addresses. 
IPv6 - supports 2^128 IP addresses.  

mac address
- physical network interface, found on motherboard. 12 character hexadecimal number. 
- first 6 are company, last 6 are unique. 
- can fake mac addresses known as spoofing. 

ping
- uses ICMP (internet control message Protocol) to determine performance of connection
- time taken is measured by ping. 
can be used by "ping ip_address"
