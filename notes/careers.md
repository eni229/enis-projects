#Careers

Security Analyst
- working with stakeholders. 
- report on safety of networks, document security issues
- security plans

Security engineer
- testing and screening security measures
- implment systems for optimal security
- monitor networks, reports to update systems. 

Incident responder - i got this in the quiz
- develop incident plan
- maintain strong security, and incident response measures.
- post incident and reports to prevent future attacks. 

Digital forensics analyst - this peaked my interest (more detective like)
- collect digital evidence while observing legal procedures
- analyse findings. 
- document findings, report on case.

Malware analyst - feel like this is self explanatory
- analyse malware, reverse engineering.
- observe activities in controlled environment
- document and report evidence

penetration tester (ethical hacker i think)
- conduct tests on computer systems, networks, and web-based applications
- perform security assessments, audits, and analyse policies
- evaluate and report on insights, recommending actions for attack prevention

red teamer (similar to previous)
- emulate role of a hacker. 
- assess organisations security controls, threat intelligence etc.
